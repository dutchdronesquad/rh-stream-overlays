# Integrations

Integrations connect Stream Overlays to external tools or data sources. They are separate from overlay themes: a theme controls the visual style, while an integration adds extra functionality.

## Available integrations

| Integration | Adds | Setup required |
|-------------|------|----------------|
| [TrackDraw](trackdraw/index.md) | Live race map and overview overlays | TrackDraw project ID and API key |

Use integration pages when you need to configure external data before adding an overlay to OBS.
