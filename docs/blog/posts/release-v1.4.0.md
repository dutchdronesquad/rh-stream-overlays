I'm excited to announce the release of v1.4.0 of the Stream Overlays! This release brings a brand new theme, improved functionality across all overlays, and a completely renewed documentation site. Let me highlight the most important changes.

<!-- more -->

## New Apex Theme

The Stream Overlays plugin now includes a completely new theme: **Apex**! This modern theme features a sleek design with topbar, node, and heat overlays. The Apex theme gives you another professional option to customize your stream's look and feel.

![Apex Theme](../../assets/img/overlays/apex/apex-heat-upcoming.gif){ style="border-radius: 5px;" }

## Upcoming Heat Overlay

All themes now support the new **upcoming heat overlay**. This feature allows you to display the next heat in your race schedule, helping viewers stay informed about what's coming up next. The overlay automatically updates as heats progress and adapts to each theme's design language.

## Renewed Documentation Site

The documentation website has been completely redesigned and expanded! The new site features:

- Modern, clean design for better readability
- Comprehensive installation and troubleshooting guides
- Improved navigation and structure
- Better organized content for finding information quickly

Visit the [documentation site](https://overlays.dutchdronesquad.nl) to explore all the new content and improvements.

## Code Quality Improvements

Behind the scenes, we've made several improvements to keep the codebase maintainable:

- Refactored CSS file structure for better organization
- Cleaned up heat overlay styles and scripts
- Updated dependencies and development tools
