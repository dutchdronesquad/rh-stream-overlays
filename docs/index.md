## Quick start guide

Ready to get started? Follow these steps:

1. **Install the plugin** — Use the [Getting Started](getting-started/index.md) guide for a 5-minute setup walkthrough
2. **Configure OBS** — Add browser sources with our [OBS setup guide](getting-started/obs-setup.md)
3. **Choose your theme** — Browse [Apex](overlays/apex.md), [DDS](overlays/dds.md), or [LCDR](overlays/lcdr.md) overlays
4. **Go live** — Everything syncs automatically with RotorHazard via Socket.IO
